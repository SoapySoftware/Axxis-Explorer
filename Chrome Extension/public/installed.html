Nice! You just installed the Axxis Explorer extension.

This will allow you to seamlessly open files on the network drive while using axxis.

Next steps

  1. Type chrome://extensions into the address bar and press enter
  2. Find the Axxis Explorer extension, and press Details
  3. Scroll down, and enable Allow access to file URLs
  4. Download the latest helper application
  5. Run the installer Axxis Explorer Installer.msi
    1. You will need to confirm that you have admin rights for the installer to run
    2. Use the default settings during the installer setup
  6. Restart Chrome
