Help: Windows helper application not properly installed

The helper application is required for the extension to function properly. Currently, the helper application is only available for machines running Microsoft Windows.

Please follow the instructions below:

  1. Download the latest helper application
  2. Run the installer Axxis Explorer Installer.msi
    1. You will need to confirm that you have admin rights for the installer to run
    2. Use the default settings during the installer setup
  3. Ensure that the extension is set up to allow local file URLs (instructions)
  4. Restart Chrome

Help: Configure extension to allow local file URLs

This setting must be manually configured.

Please follow the instructions below:

  1. Type chrome://extensions into the address bar and press enter
  2. Find the Axxis Explorer extension, and press Details
  3. Scroll down, and enable Allow access to file URLs
  4. Restart Chrome

Help: When clicking a local file link, multiple tabs open

Most likely you have multiple extensions installed that are performing similar actions.

Please follow the instructions below to disable other applications:

  1. Type chrome://extensions into the address bar and press enter
  2. Find any conflicting extensions, and disable them. For example, Enable local file links and Local Explorer
