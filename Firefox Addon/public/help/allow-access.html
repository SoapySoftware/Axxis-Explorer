Help: Configure extension to allow local file URLs

This setting must be manually configured.

Please follow the instructions below:

  1. Open a new tab, and type about:addons into the address bar and press enter
  2. Find and click on the Axxis Explorer add on
  3. Click Permissions and then enable the permissions to allow the add-on to access data for marconmetalfab.com and axx.is
  4. Restart Firefox
